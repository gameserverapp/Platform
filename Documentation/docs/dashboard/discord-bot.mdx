The GSA Discord bot adds features like `!kickme` commands for players and game server management commands for the Admin team. Talk with in-game players from Discord and more.

[Go to Discord bot settings >](https://dash.gameserverapp.com/configure/discord)

## Bot commands
Discord users should first connect their Discord to GSA in order for the bot to recognize them.

## Features

:::info Channel update delay
It may take a few minutes for new Discord channels to show up in the dropdown lists.
:::

### Discord + Cluster chat
Talk directly with in-game players from Discord and visa versa. All games that [support Cluster chat](https://dash.gameserverapp.com/features-per-game) can use this feature.

![Discord bot - Cluster chat + Discord](/img/dashboard/discord_bot/cluster_chat_discord_chat_example.jpg)

Navigate to the [Discord page](https://dash.gameserverapp.com/configure/discord) and select a Discord channel for the `Discord + Cluster chat` field.


:::info
After saving the Discord channel it may take a couple minutes for the Discord + Cluster chat to start working. Please be patient.
:::

![Discord bot - Enable Cluster chat + Discord](/img/dashboard/discord_bot/enable_discord_cluster_chat.jpg)



### Player join / leave
Report join / leave messages to Discord and see when a new player joins a game server.

![Discord bot - Player join / leave](/img/dashboard/discord_bot/player_join_leave.jpg)

Navigate to the [Discord page](https://dash.gameserverapp.com/configure/discord) and select a Discord channel for the `Player join & leave` field.

![Discord bot - Player join / leave](/img/dashboard/discord_bot/discord_player_join_leave_setup.jpg)


### Game server status card

Shows a status card that is automatically updated on events, like updates or scheduled restarts.

![Discord bot - Game server status card](/img/dashboard/discord_bot/discord_game_server_status_card.jpg)

Navigate to the [Discord page](https://dash.gameserverapp.com/configure/discord) and select a Discord channel for the `Game server status card` field.

![Discord bot - Player join / leave](/img/dashboard/discord_bot/discord_game_server_status_card_setup.jpg)

### Game server status log

Shows detailed activity about game servers, like when a mod update was detect or the game server is starting up / shutting down.

![Discord bot - Game server status log](/img/dashboard/discord_bot/discord_game_server_status_log.jpg)

Navigate to the [Discord page](https://dash.gameserverapp.com/configure/discord) and select a Discord channel for the `Game server status log` field.

![Discord bot - Player join / leave](/img/dashboard/discord_bot/discord_game_server_status_log_setup.jpg)

### Group logs to Discord
Let players report their group logs to a private Discord server, so they can see what is happening in-game when they're offline.



## How-to

### Invite bot to your Discord
In order to use the Discord bot, it should first be connected with the Discord server.

#### 1. Connect Discord
Navigate to the [Discord page](https://dash.gameserverapp.com/configure/discord) and hit `Connect your Discord`.

![Discord bot - Connect discord 1](/img/dashboard/discord_bot/connect_discord_1.jpg)

#### 2. Select Discord server
Select the Discord server the bot should join and click `Continue`.

![Discord bot - Connect discord 2](/img/dashboard/discord_bot/connect_discord_2.jpg)

#### 3. Confirm permissions
Leave all listed permissions enabled as the bot will need all of them.
Click `Authorize` to invite the bot.

![Discord bot - Connect discord 3](/img/dashboard/discord_bot/connect_discord_3.jpg)

#### 4. Done!

![Discord bot - Connect discord 4](/img/dashboard/discord_bot/connect_discord_4.jpg)



### Connect Discord for players
Learn how to connect your Discord account with GSA as a player.

#### 1. Click connect link
Type `!help` in a channel on the community Discord (the one you're playing on) to bring up the connect link.

![Discord bot - Connect player 1](/img/dashboard/discord_bot/discord_connect_player_1.jpg)

#### 2. Authorize Discord
Click `Authorize` to give the bot your Discord ID and username to identify you on Discord.

![Discord bot - Connect player 2](/img/dashboard/discord_bot/discord_connect_player_2.jpg)

#### 2. Success!
After successfully connecting your Discord it may take a few minutes for the bot to recognize you.

![Discord bot - Connect player 3](/img/dashboard/discord_bot/discord_connect_player_3.jpg)

### Setup group logs for communities
Learn how to setup Group logs so your group can access group logs on your private Discord.

#### 1. Log in on Community website
Group logs can be enabled per group, by the group admins. In order to do this the group admin should log in on your Community website.

![Discord bot - Group log setup 1](/img/dashboard/discord_bot/group_log_setup_1.jpg)

Head over to `Group settings`.

![Discord bot - Group log setup 2](/img/dashboard/discord_bot/group_log_setup_2.jpg)

#### 2. Connect group Discord
On the Group settings page click `Connect Discord` to invite the bot to your private Discord server.

![Discord bot - Group log setup 3](/img/dashboard/discord_bot/group_log_setup_3.jpg)

Select the Discord server that should have the group logs.

![Discord bot - Group log setup 4](/img/dashboard/discord_bot/group_log_setup_4.jpg)

Leave all listed permissions enabled as the bot will need all of them.
Click `Authorize` to invite the bot.

![Discord bot - Group log setup 5](/img/dashboard/discord_bot/group_log_setup_5.jpg)

#### 3. Select Discord channel
Select the Discord channel where you want the group logs.

![Discord bot - Group log setup 6](/img/dashboard/discord_bot/group_log_setup_6.jpg)

#### 4. Done!
New group logs are now automatically reported to the configured Discord channel. It may take up to 10 minutes for new group logs to post on your Discord.

![Discord bot - Group log setup 7](/img/dashboard/discord_bot/group_log_setup_7.jpg)

Make sure the bot has permissions to talk in all channels. Check the bot roles and ensure it can send messages.
